# 🚀 SwingBy (스윙바이)

**SwingBy**는 궤도를 설계하는 우주 물리 퍼즐 게임입니다.

## 🎮 게임 실행 방법
1. `Build` 폴더 내의 모든 파일을 다운로드합니다.
2. `SwingBy.exe`를 실행합니다.
3. 터미널 창의 `>` 기호 뒤에 명령어를 입력하여 게임을 제어할 수 있습니다.
   - `/start`: 스테이지 시작
   - `/exit`: 게임 종료
   - `/help`: 명령어 보기

## 🕹️ 조작법
| 키 | 기능 |
| :--- | :--- |
| **W, S / A, D / Q, E** | 플레이어 캐릭터 회전 |
| **Spacebar** | 가속 (캐릭터의 빨간색 돌출부 반대 방향으로 추진) |
| **C** | 카메라 모드 전환 (캐릭터 고정 ↔ 카메라 위치 고정 시점) |
| **R** | 게임 중 터미널 화면으로 복귀 |
| **마우스 우클릭** | 카메라 고정 모드 시 캐릭터 주변 회전 |

## 💡 게임 플레이 팁
- **목표**: 맵 어딘가에 위치한 거대한 핑크색 구체를 찾아 도착하면 게임이 종료되고 터미널로 돌아갑니다.
- **길 찾기**: 우측 상단의 `Distance` 수치를 확인하며 핑크색 구체를 찾아보세요.
- **카메라 모드**: `C`를 누르면 캐릭터를 따라가던 카메라가 해당 위치에 고정됩니다. 다시 `C`를 누르면 고정된 위치에서 카메라가 해제됩니다.
- **터미널 인터페이스**: `/help`와 같은 명령어를 사용하면 `>` 기호가 사라질 수 있습니다. 이는 커서가 사라진 것이 아니므로, 기존 입력창 위치를 클릭하면 다시 입력을 이어갈 수 있습니다.

## ⚠️ 현재 이슈
- **에셋(Asset) 미적용**: 현재 개발 초기 단계로, 대부분의 오브젝트가 회색 구체 형태로 표현됩니다.
- **성능**: 오브젝트가 많아 사양에 따라 다소 렉이 발생할 수 있습니다.
- **터미널**: 터미널에서 /help같은 명령어를 너무 많이 입력하면 터미널이 상단에 고정되는 이슈가 있는데 /clear로 다시 정리할 수 있습니다.

## 📅 업데이트 계획
- **스테이지 추가**: 현재 플레이 가능한 스테이지 외에, 더욱 복잡한 궤도 설계가 필요한 새로운 스테이지들이 지속적으로 업데이트될 예정입니다.
